我：问我不如问百度，百度它不香吗🤔

慢扫描电视（Slow-scan television）是业余无线电爱好者的一种主要图片传输方法，慢扫描电视通过无线电传输和接收单色或彩色静态图片。慢扫描电视的一个术语名称是窄带电视。广播电视需要 6MHz 的带宽，因为帧速为 25 到 30fps 。慢扫描电视的带宽只有 3kHz ，因此慢扫描电视是一种慢得多的静态图像传输方法，通常每帧需要持续 8 秒或若干分钟。业余无线电操作员通常在短波（或高频）、甚高频、超高频波段使用慢扫描电视。

恰好我对无线电有点兴趣，就利用空闲时间研究了一下 SSTV ，那就从 通过声音信号发送图片 的经历开始说起吧， 废话不多说 ，上步骤🤠：

1.搜索 MMSSTV 的官网并下载

MMSSTV 官网 MMSSTV官网。

2.安装好后打开软件

3.点击 Template

4.鼠标右键点击

5.选中 delete ，清空点击 Template 界面的内容（因为原有界面可能有初始图片）

6.复制你想要发送的图片

7.用鼠标右键点击 Template 界面

8.选择 Paste image

9.选择以点阵发送

10.点击 TX

11.选择 TX Mode （如 Robot 72 ，用它解析出来的图片质量比较好😄）

12.在手机百度搜索 Robot 36 并下载（Robot 36 是手机软件哦😆）

13.安装好后打开软件

14.点击上方菜单栏有着三个点的按钮

15.选择 Select Mode

16.选择之前选择的 TX Mode 类型（如 Robot 72）

17.点击开始按钮开始解析声音

18.点击电脑 MMSSTV 的 TX 界面红色的 TX 通过声音信号发送图片

电脑开始发出诡异的声音🫨

19.手机的麦克风一定要靠近电脑的喇叭（不然接收到的图片会很模糊）

20. MMSSTV 的图片音频播放完毕后，在手机的 Robot 36 会显示解析出来的图片
